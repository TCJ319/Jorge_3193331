Información sobre el Proyecto

Este proyecto tiene como objetivo proporcionar un formulario simple para recoger datos de los usuarios, como su nombre y color favorito. La página de información te permite conocer más sobre el propósito del proyecto y cómo utilizarlo.

Además, este proyecto es una excelente manera de practicar HTML y CSS, combinando la estructura de las páginas con un diseño atractivo.

Puedes navegar entre el formulario y esta página de información para explorar las diferentes características.
